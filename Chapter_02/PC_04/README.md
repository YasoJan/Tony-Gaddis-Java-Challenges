# 4. Star Pattern

## Problem
Write a program that displays the following pattern:

   * 
  ***
 ***** 
******* 
 ***** 
  ***
   *

## Variables
This program uses no variables — the pattern is passed directly as a string 
literal to a dialog box.

## Expected Output
A `JOptionPane` message dialog displaying the diamond-shaped star pattern.

## Concepts Practiced
- Importing classes from the Java standard library (`javax.swing.JOptionPane`)
- Displaying GUI message dialogs with `JOptionPane.showMessageDialog()`
- Using `\n` escape sequences to create multi-line strings
- Calling `System.exit(0)` to cleanly terminate a program after a Swing dialog
- String literal formatting to produce visual patterns
