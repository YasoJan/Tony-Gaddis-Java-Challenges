# Programming Challenge 3: Personal Information

## Problem
Write a program that displays the following information, each on a separate line:
- Your name
- Your address, with city, state, and ZIP
- Your telephone number
- Your college major

Although these items should be displayed on separate output lines, use only a 
single `println` statement in your program.

## Variables
| Variable | Type | Description |
|---|---|---|
| `name` | `String` | Full name |
| `address` | `String` | Street, city, state, and ZIP |
| `telephone` | `String` | Phone number |
| `collegeMajor` | `String` | Field of study |

## Expected Output

-------------- Personal Information --------------
Name: Yasin Zahir
Address: 77777 Golden Rd, Utopia, California, 77777
Telephone: +7 (777) 777-7777
College Major: Computer Science (AI)
-------------- Personal Information --------------

## Concepts Practiced
- Using the `\n` escape sequence to create newlines within a single string
- Producing multi-line output from a single `println` call
- String concatenation with the `+` operator
- Storing structured personal data in `String` variables
